
<p>Web application to analyze Federal Regulations within the Electronic Code of Federal Regulations (eCFR). This site will assist in making potential deregulation efforts across the government.</p>
<h2>Tech Stack</h2>
<ul>
  <li><b>Frontend:</b> ColdFusion 2023</li>
  <li><b>Backend/Datasource:</b> SQL Server 2025, eCFR API endpoints</li>
  <li><b>UI:</b> Datatables Javascript Library</li>
                       
</ul>
